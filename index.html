Configuração inicial

Para sincronizar os dados na nuvem gratuitamente, você precisará de um projeto Firebase. Siga os passos abaixo uma única vez.

Passo 1 de 3 — Criar projeto Firebase
  1. Acesse console.firebase.google.com
  2. Clique em "Adicionar projeto", dê um nome e crie
  3. No menu lateral, clique em "Firestore Database"
  4. Clique em "Criar banco de dados", escolha "Modo de teste"
  5. Escolha a região southamerica-east1 (São Paulo)
Passo 2 de 3 — Credenciais
  1. No painel do Firebase, clique em "Adicionar app" → ícone </>
  2. Dê um apelido e clique em "Registrar app"
  3. Copie os valores do bloco firebaseConfig
Credenciais salvas apenas no seu navegador. Nunca enviadas a terceiros.
Passo 3 de 3 — Regras de segurança

No Firebase Console → Firestore → Regras, cole:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
Modo aberto seguro para uso local. Configure autenticação se precisar de acesso remoto.
